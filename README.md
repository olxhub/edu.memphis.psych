# Memphis Introductory Psychology

A scenario-based introduction to behavioral psychology, developed at the
University of Memphis for an introductory (freshman) psychology course. Students
work as a "study group" (the *Psychology Study Group* SBA series) helping a
parent through a running case, learning the concepts by applying them — each
activity is an interactive SMS-style conversation paired with the ideas it
teaches.

The teaching unit is the **scenario-based learning activity (SBLA)**. A full
sequence runs through four parts:

1. Parenting Techniques
2. Operant Conditioning
3. Function of Behavior
4. Fidelity of Implementation

## Scenarios

The four-part structure is reused across interchangeable *scenarios* — the
psychology stays fixed while the story changes:

- **Sleep refusal** (`psychology/`) — the original case: a child who won't go to
  bed. It began as two parts and grew to all four; Parts 3–4 are currently held
  back for the fall term.
- **Child defiance** (`psychology/defiance/`) — a parallel case with the same
  structure: a child who won't clean up their room. Parts 1–2 so far.

Each scenario is its own content namespace (`edu.memphis.psych` and
`edu.memphis.psych.defiance`), so the two sit side by side without colliding and
launch independently.

## Direction

The aim is to grow toward a full set of activities for an entire freshman
course — more scenarios, and more parts within each — built on the established
four-part SBLA structure. Specific goals vary by staff member.
